# teacher
基于vuejs的课堂辅助系统，结合微信小程序端进行对学生上课的一些辅助操作。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files

```
npm run lint
```
